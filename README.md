# Py1010 – Prosjektoppgave

Dette repositoryet inneholder min besvarelse på arbeidskrav i emnet Py1010.

 Innhold

- `Prosjektoppgave.ipynb` – hovedbesvarelse
- `hendelser.csv` – datasett brukt i oppgaven
- `opprettDatabaseOgTabeller.sql` – SQL-script for databaseoppsett

 Beskrivelse

Oppgaven går ut på å hente data fra NVDB API, behandle dataene i Python og lagre dem i en SQL Server database, hvorpå jeg spør og visualiserer noe data.
